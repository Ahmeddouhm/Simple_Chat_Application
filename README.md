# Simple Socket Programming Chat Application
**Overview:** A real-time chat system for Windows that uses a Client-Server architecture.
**How It Works:** A central Server program listens for connections on a specific port. Multiple Client programs connect to this server to send and receive messages instantly.
**Technology:** Built using native C++ with the Winsock2 library for networking and the Win32 API for the graphical user interface.
**Key Features:** It uses multi-threading to handle multiple users simultaneously without freezing and features a custom dark-themed interface.
